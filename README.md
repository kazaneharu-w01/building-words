# 現場作業者のための建築用語辞典 v2

`building_words` と同じ仕様・同じデータ (`terms_data.json`) のまま、UI/UXを刷新したバージョン。

## 起動方法

静的サイトなので、任意のHTTPサーバーで配信するだけで動く。

```sh
python -m http.server 8741
# → http://localhost:8741
```

※ `fetch` でJSONを読むため、`file://` 直開きでは動かない。

## ページ構成

仕様書どおりの3ページ構成に整理した(旧版の categories.html / category.html は search.html のカテゴリフィルタに統合)。

| ファイル | 役割 |
|---|---|
| `index.html` | トップ。ヒーロー検索(サジェスト付き)、人気ワード、カテゴリ、ピックアップ、ガイド |
| `search.html` | 用語一覧。`?q=` キーワード検索 + `?cat=` カテゴリ絞り込み。入力中の即時絞り込み・ヒット箇所ハイライト対応 |
| `term.html` | 用語詳細。`?term=用語名&cat=カテゴリID`。意味/場面/注意点/関連用語を表示 |

## 旧版からの主な改善点

- **デザイン刷新** — ネイビー×セーフティイエローのトーンは維持しつつ、Noto Sans JP・ブループリント風グリッドのヒーロー・カードUIで全面再構成
- **検索サジェスト** — トップの検索バーに入力候補ドロップダウン(キーボード操作対応)
- **即時絞り込み** — 一覧ページは入力のたびに結果を更新し、一致箇所を `<mark>` でハイライト。用語名・読み一致を上位に並べ替え
- **カテゴリフィルタ** — チップUIで一覧ページ内からワンタップ切り替え。URLにも反映されるので共有可能
- **アイコン** — 漢字1文字の擬似アイコンをやめ、カテゴリごとのインラインSVGアイコンに変更
- **モバイル対応** — ハンバーガーメニュー、横スクロールするフィルタチップ、1カラムカード
- **アクセシビリティ** — スキップリンク、`aria-current` / `aria-pressed` / `role="listbox"`、focus-visible スタイル、`prefers-reduced-motion` 対応

## データ

`terms_data.json` は旧版と同一(14カテゴリ・390語)。スキーマは `../building_words/data_schema.md` を参照。
